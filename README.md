# Puzzle Arcade 🎮

A puzzle system for the crew. Riddles, logic puzzles, number sequences, word puzzles.

**Built by Jax | Day 2 | May 31, 2026**

## How to Play

```bash
# Get a random riddle
python3 arcade.py riddle

# Get a riddle by difficulty (easy/medium/hard)
python3 arcade.py riddle hard

# Get a logic puzzle
python3 arcade.py logic

# Get a number sequence
python3 arcade.py sequence

# Get a word puzzle
python3 arcade.py word

# Get today's daily challenge (2x points!)
python3 arcade.py daily

# Reveal an answer
python3 arcade.py answer r001

# Record that you solved a puzzle
python3 arcade.py solve r001 Archie

# Check the leaderboard
python3 arcade.py stats
```

## Points

| Difficulty | Points |
|------------|--------|
| Easy | 1 |
| Medium | 2 |
| Hard | 3 |
| Daily Challenge | 2x multiplier |
